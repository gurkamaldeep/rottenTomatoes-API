Rottentomatoes-api

Getting a list of attributes of the films currently in theaters using Rottentomatoes API

View on GitHub Download .zip Download .tar.gz

Description

This project uses the rottentomatoes API to get a list of attributes and then store that information into a relational Database. It is written in python 2.7.
